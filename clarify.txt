Project file audit — relevance of every file/folder in the project root
=====================================================================

ESSENTIAL (keep — actively used by the build or server):

  .eleventy.js          Eleventy config. Defines input/output dirs and passthrough copies.
  .gitignore            Git ignore rules.
  assets/icons/         SVG icons used in the site (phone, mail, clock, location, etc.).
  assets/logo/          Logo images in PNG + WebP formats (home, negative, favicon, horizontal, square).
  CLAUDE.md             Instructions for Claude Code.
  ENV.md                Docs for MailerSend environment variables.
  fonts/NeueRegrade/    The 13 Neue Regrade font files actually used in style.css @font-face declarations.
  header.js             Nav menu + sticky header JavaScript.
  package.json          Dependencies and scripts.
  package-lock.json     Lockfile.
  render.yaml           Render.com deployment config.
  script.js             Contact form handler JavaScript.
  server.js             Node.js Express server with MailerSend integration.
  src/                  4 Nunjucks page templates (index, sobre, servicos, contacto) + header/footer partials.
  style.css             Main stylesheet (~1050 lines).

LEGITIMATE (not core, but harmless):

  .DS_Store             macOS desktop service file. Safe to delete but will reappear.
  .agents/              Claude Code agent/skill definitions. Not critical.
  .claude/              Claude Code local settings. Not critical.
  skills-lock.json      Claude Code skill lock file. Generated automatically.
  vercel.json           Leftover from an earlier Vercel deployment attempt. Not used (deployed on Render).
  .env                  Local environment variables (gitignored). Keep for local dev.

TRASH (safe to delete — duplicates or unused):

  fonts/Nova pasta/     Exact duplicate of fonts/NeueRegrade/ (same 13 files). "Nova pasta" = "New folder" in PT.
  hipoteses-validas-site@1.0.0   Looks like a stray file or artifact. Not used by anything.

NOTES ON LOGO FILES (assets/logo/):

There are multiple formats and sizes. The templates reference these:
  - logo-hv-home-optimized.png and logo-hv-home.webp (via srcset in header)
  - logo-hv-home-neg-optimized.png and logo-hv-home-neg.webp (via srcset in footer)
  - logo-hv-favicon-optimized.png (favicon)

These are NOT referenced anywhere in templates and may be unused:
  - logo-hv-home.png, logo-hv-home-neg.png
  - logo-hv-horizontal.png, logo-hv-horizontal.webp, logo-hv-horizontal-optimized.png
  - logo-hv-square.png, logo-hv-square.webp, logo-hv-square-optimized.png
  - logo-hv-favicon.png, logo-hv-favicon.webp
